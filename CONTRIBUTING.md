# Contributing to Energy Credit (ENE)

Thank you for your interest in contributing!  
This repository focuses on documentation, whitepapers, and assets related to ENE.

## How to contribute
- Report issues or ideas via [GitHub Issues](../../issues)
- For small fixes, fork the repo and open a Pull Request
- For proposals, please open an Issue first for discussion

## Guidelines
- Keep PRs concise and focused
- Use clear commit messages (e.g., `docs: update whitepaper reference`)
- Make sure all external links in the README are valid
