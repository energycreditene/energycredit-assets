# Security Policy

If you discover a vulnerability, please email **security@energycredit.pro**.

We appreciate responsible disclosure and will work to address issues promptly.

- Do **not** open public GitHub Issues for security reports.
- Provide as much detail as possible, including steps to reproduce.
- Once verified, the core team will acknowledge receipt and provide status updates.

---

Thank you for helping keep Energy Credit (ENE) secure and transparent.
